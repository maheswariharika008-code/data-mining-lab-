Data Mining Lab

Description

This repository contains Data Mining lab programs implemented using Python. The experiments focus on extracting meaningful insights from datasets using various data mining techniques.

Objectives

- To understand data preprocessing techniques
- To apply classification and clustering algorithms
- To analyze datasets and visualize results
- To evaluate model performance

Tools & Technologies

- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

List of Programs

1. Data Preprocessing
2. Data Visualization
3. Classification using Decision Tree
4. Classification using Naive Bayes
5. K-Nearest Neighbors (KNN)
6. Linear Regression
7. Logistic Regression
8. K-Means Clustering
9. Hierarchical Clustering
10. Association Rule Mining (Apriori Algorithm)
